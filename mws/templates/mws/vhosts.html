By default, each MWS server contains a default website associated with the hostname of the MWS server. This means that you can start using your MWS server without further configuration. Web site contents should be stored inside /var/www/{name_of_website} folder of your MWS server.

You can create as many web site as you want in you MWS server but be aware that they will share your MWS server resources. You need to use different names for each website.

Once you have created you website you will need to associate a domain name to it using the option buttons on the right in order to access it.
