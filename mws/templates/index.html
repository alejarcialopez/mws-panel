This site allows members of the University to request a new server in the Managed Web Service. We provide users with a web-server that they can use without having to worry about manual configuration or operating system maintenance. A server in the Managed Web Service has the following characteristics:

  • A server with {{ features.numcpu }} Core CPU, {{ features.sizeram }} GB of RAM, and {{ features.sizedisk }}Gb of disk space (only 50% is available to the user, the rest is used for other purposes like snapshots)
  • A public IPv4 and IPv6 addresses
  • A simple web-based control panel to control all the features of the Managed Web Service
  • A cost of only Â£{{ features.price }} a year

After you have registered your new server, you can then set up one or more websites on it.

Warning: If you don't enter billing details for your site within a month, it may be deleted.

You will get a Debian Jessie (8) box with (between others) the following software installed by default:

  • Apache 2.4
  • PHP 5.6
  • Python 2.7
  • Perl 5.20
  • MySQL 5.5
